#Olá! Essa é a edição PC do app_fii
#Essa edição contém uma gui amigável ao usuário fácil de usar.
#
#INSTALAÇÃO
#
#1) Garanta que o seu desktop tenha pelo menos dois espaços disponíveis.
#
#2) Vá no seu desktop, clique com o botão direito e vá em 'novo'. De lá, clique em 'arquivo de texto', nomei o novo arquivo como
#'meus_fundos'.
#
#3) Clique duas vezes sobre o arquivo e quando ele abrir, escreva os nomes dos seus fundos separados por ';' todos na mesma linha,
#conforme o exemplo abaixo:
#
#alzr11;bcpf11;mxrf11
#
#4) Vá em: "https://github.com/joaopedrolourencoaffonso/auto_fii_app/edit/app_fii_pc/PC" e faça download do arquivo 'app_fii.exe'
#
#5) Salve o arquivo no seu DESKTOP, se você colocar em outro diretório, o app_fii não vai funcionar.
#
#6) Pronto, o app_fii está preparado para rodar!
#
#OBSERVAÇÕES
#Dentro do próprio auto_fii existe uma seção 'Instruções', mesmo assim, gostaria de descrever os seguintes pontos do projeto:
#1) Para registrar os seus fundos, o app_fii faz uso do arquivo "meus_fundos.txt que você criou, NÃO o exclua ou você vai ter que registrar os fundos novamente!

#2) O app_fii retira as informações dos sites "meusdividendos.com" e "fii.com", se um desses sites não estiver funcionando, o app_fii
#NÃO VAI FUNCIONAR
#
#3) O app_fii foi projetado para funcionar apartir do desktop, SALVE O APP_FII NO DESKTOP, se você não o fizer, há o risco do app_fii NÃO FUNCIONAR
#
#4) Essa é apenas a versão inicial do app_fii, no futuro, pretendo fazer uma versão mais completa e fácil de usar.
